# Sustainability Reports — NEXA

Repositorio de los Relatórios de Sustentabilidade de la **Nexa SGOIC, S.A.**, sociedade gestora de organismos de investimento coletivo regulada pela CMVM (Portugal).

## Estructura

```
sustainability-reports-NEXA/
├── pdfs/                       # PDFs originales de los relatórios
│   └── RS_NEXA_2025.pdf
└── data/                       # Datos estructurados en JSON
    ├── index.json              # Índice de archivos
    ├── metadata.json           # Metadatos del informe
    ├── company.json            # Info de la empresa
    ├── board-message.json      # Mensaje del Board
    ├── funds.json              # 3 fondos bajo gestión
    ├── assets.json             # 18 activos imobiliarios
    ├── people.json             # Equipo y departamentos
    ├── materiality.json        # Análisis de Doble Materialidad
    ├── esg-targets.json        # Métricas y metas ESG
    ├── environmental.json      # Métricas ambientales
    ├── social.json             # Métricas sociales
    ├── governance.json         # Gobernanza
    ├── economic.json           # Métricas económicas
    ├── future-perspectives.json # Perspectivas futuras
    ├── final-considerations.json
    └── glossary.json           # Glosario de términos
```

## Relatório 2025

- **Período**: 1/01/2025 – 31/12/2025 (comparativo con 2024)
- **Ativos sob gestão**: 198 M€ — 18 ativos imobiliários (incluyendo landbank)
- **Cidades**: 8 — Portugal
- **Colaboradores**: 10
- **Fondos**: Icon SIC, Fuse Valley FII, Castro Capital SIC
- **Primera certificación LEED Gold**: edificio La Movida

## Uso

Los archivos JSON están pensados como base de datos ligera para alimentar una página web (próximo paso del proyecto).
